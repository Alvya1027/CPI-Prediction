# 严格封闭50窗口：单光储备池与孪生光储备池对比

实验日期：2026-07-23  
运行环境：Python 3.11.9  
实验性质：独立小样本探索实验，不覆盖原第三阶段及recent50开放参考库结果  
评估口径：固定50窗口训练数据池；固定50窗口历史参考库

## 1. 实验目的

本实验回答以下问题：

> 当单光储备池和孪生光储备池能够访问的训练数据总量都严格限制为相同的50个月时，孪生差值回归是否仍然优于单网络直接回归？

这里不再允许孪生模型调用50个月之前的136个历史参考窗口。单网络和孪生模型能够访问的训练数据池完全相同，均为原始时间顺序训练集最后50个目标窗口。

## 2. 严格封闭规则

训练数据池固定为：

| 项目 | 数值 |
| --- | --- |
| 可访问训练窗口 | 50 |
| sample_id | 162–211 |
| 目标月份 | 2014-07至2018-08 |
| 状态维度 | 50 |

孪生模型强制满足：

1. 训练目标 `sample_i_id` 必须属于162–211；
2. 训练参考 `sample_j_id` 必须属于162–211；
3. 验证集只能从这50个训练窗口中选择参考；
4. 测试集也只能从这50个训练窗口中选择参考；
5. 不允许使用sample_id 0–161的更早训练窗口；
6. 不允许使用验证集真实标签作为后续参考；
7. 不允许使用较早测试月份真实标签作为后续参考；
8. 目标与参考仍保持原来的12个月隔离规则；
9. 验证和测试参考选择仍不使用目标真实CPI。

自动检查结果：

```text
training_reference_ids_all_inside_closed_pool = true
validation_reference_ids_all_inside_closed_pool = true
test_reference_ids_all_inside_closed_pool = true
older_train_references_used = false

```

## 3. 为什么孪生模型只能形成27个配对目标

训练池虽然有50个窗口，但孪生训练要求参考窗口必须更早，并且目标窗口开始月份与参考窗口结束月份至少间隔12个月。

因此，训练池最前面的23个窗口找不到合法的更早参考，只能作为后续窗口的历史参考，不能作为具有合法参考的 `sample_i` 配对目标。

最终得到：

| 项目 | 数值 |
| --- | ---: |
| 总可访问训练窗口 | 50 |
| 合法候选训练对 | 378 |
| 具有合法参考的目标 | 27 |
| 分层筛选后的训练对 | 118 |
| 没有合法更早参考的窗口 | 23 |

这不是删除了23个月数据。这23个月仍属于50窗口数据池，并可以作为历史参考，只是不能在当前12个月隔离规则下充当孪生配对的目标端。

两种模型的训练形式为：

```text
单网络：
50个窗口 → 50条 h_i → CPI_i 直接回归记录

孪生模型：
同一个50窗口池 → 27个合法配对目标 → 118条
h_i - h_j → CPI_i - CPI_j 关系记录
```

## 4. 保持不变的设置

- 单光储池状态使用同学 `optical_reservoir_recent50_20260723` 的状态缓存；
- 孪生模型使用同一份状态缓存；
- 训练、验证、测试状态与原Simulink状态逐元素差值为0；
- 输入窗口长度为12个月；
- 状态维度为50；
- 参考策略为 `window_distance`；
- 验证和测试每个目标使用 `K=10`；
- 状态特征为 `h_i-h_j`；
- 聚合方式为距离倒数加权；
- Ridge alpha候选范围不变；
- alpha只根据验证集RMSE选择。

单网络与严格封闭孪生模型最终均选择：

```text
alpha = 1
```

## 5. 正式结果

| 模型 | 可访问训练窗口 | 直接训练目标 | 孪生配对目标 | 训练对 | 验证 MAE | 验证 RMSE | 测试 MAE | 测试 RMSE |
| --- | ---: | ---: | ---: | ---: | ---: | ---: | ---: | ---: |
| 单光储备池 closed50 | 50 | 50 | 不适用 | 不适用 | 0.582056 | 0.781584 | 0.910261 | 0.988836 |
| 孪生光储备池 closed50 | 50 | 不适用 | 27 | 118 | **0.516337** | **0.696738** | **0.799978** | **0.886371** |

孪生模型相对单网络：

- 验证MAE降低11.29%；
- 验证RMSE降低10.86%；
- 测试MAE降低12.12%；
- 测试RMSE降低10.36%。

## 6. 残差分析

| 模型 | 测试平均残差 | 残差标准差 | 中位绝对误差 | 最大绝对误差 |
| --- | ---: | ---: | ---: | ---: |
| 单网络 closed50 | 0.7980 | 0.5839 | **0.9152** | 1.7382 |
| 孪生 closed50 | **0.7153** | **0.5235** | 0.9410 | **1.5439** |

孪生模型降低了整体高估、残差波动、MAE、RMSE和最大误差，但中位绝对误差略高于单网络。这说明其改善不是每个月都一致，而是主要减少了部分较大的误差。

严格封闭孪生模型误差较大的月份包括：

| 目标月份 | 绝对误差 |
| --- | ---: |
| 2025-02 | 1.5439 |
| 2024-11 | 1.4363 |
| 2023-10 | 1.3437 |
| 2022-06 | 1.2489 |
| 2025-09 | 1.2359 |

两个模型在测试中仍然存在明显高估。这与50个月训练区间的CPI水平及测试期分布变化有关。

## 7. 与开放历史参考库实验的区别

| 模型/口径 | 训练数据池 | 历史参考库 | 孪生配对目标 | 测试 MAE | 测试 RMSE |
| --- | ---: | ---: | ---: | ---: | ---: |
| 单网络 recent50 | 50 | 无 | 不适用 | 0.910261 | 0.988836 |
| 孪生 strict closed50 | 50 | 同一50窗口 | 27 | 0.799978 | 0.886371 |
| 孪生 recent50开放参考库 | 50个监督目标 | 136个额外历史参考 | 50 | 0.320615 | 0.390343 |

开放参考库时孪生结果大幅提升；严格关闭额外历史数据后，优势缩小到约10%–12%。

这说明此前的大幅提升同时来自两个部分：

1. 孪生状态差和历史锚点机制本身；
2. 额外的历史参考库提供了更丰富的CPI水平和变化关系。

严格封闭实验更适合用于回答“相同总数据预算下孪生结构是否有效”。结果显示它仍有一定改善，但改善幅度有限，不能再声称提高60%左右。

## 8. 客观结论

在两种模型能够访问的训练数据总量都严格限制为相同50个月时：

> 孪生光储备池的测试MAE相对单光储备池降低12.12%，测试RMSE降低10.36%，说明共享光储备池状态差和参考式回归在相同总数据预算下仍然具有一定价值。

但必须同时说明：

- 孪生模型由于时间隔离规则只有27个合法配对目标；
- 两种模型都存在明显测试期高估；
- 孪生模型中位绝对误差没有改善；
- 当前测试集此前已经被查看，本实验属于探索性复测，不是独立盲测；
- 若要进一步调整间隔、特征或参考数量，应重新进行验证设计，并使用新增CPI月份进行外部测试。
