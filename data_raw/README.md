# 原始数据说明

本文件夹存放 CPI 月度预测项目的原始数据文件。后续清洗脚本会从这里读取数据，并生成 `data_processed/` 下的可建模数据集。

## 文件

- `cpi_raw.xlsx`：原始 CPI 月度数据表，包含数据本体和来源说明。

## Excel 工作表

- `CPI数据`：CPI 月度观测值。
- `来源说明`：数据来源、时间范围和可信度说明。
- `图表数据`：用于画图或检查的月度序列副本。

## 主要字段

- `月份`：月份，原始格式主要是 `YYYYMM`。
- `CPI同比增长率(%)`：CPI 同比增长率，单位是百分点。
- `居民消费价格指数(上年同月=100)`：CPI 同比指数，以上年同月为 100。

## 数据来源与可信度

- `2026-04` 至 `2008-01`：来自已有 CPI 月度数据文件。
- `2007-12` 至 `2006-01`：来自世铝网数据中心网页表格，网页注明数据来源为国家统计局。
- `2005-12` 至 `2000-01`：根据 MacroMicro 折线图进行像素读数估算。这一段可以用于第一周流程跑通和 baseline 对比，但正式论文或汇报前建议用国家统计局、Wind、CEIC 等更权威来源替换或核验。

## 清洗脚本

在项目根目录运行：

```bash
python src/clean_cpi_data.py
```

运行后会生成：

```text
data_processed/cpi_monthly.csv
```
