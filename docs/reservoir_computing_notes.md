# 储备池计算（Reservoir Computing）笔记

> 本文档面向项目组所有成员，包含概念速览和公式推导两个层次。速览部分用通俗语言讲清楚"是什么"和"为什么"，推导部分为后续实现做准备。

---

## 1. 概念速览

### 1.1 一句话解释

储备池计算是一种处理时间序列的神经网络方法。它的核心思路是：用一个随机固定的大网络（储备池）把输入信号"搅乱"到高维空间，然后只训练最后一步的线性读出层。储备池本身不训练。

### 1.2 为什么储备池内部不训练？

传统 RNN 训练困难，因为误差要沿时间反向传播，计算量大、容易梯度消失。RC 绕过了这个问题：

- 储备池的连接权重随机生成后就固定了
- 输入数据经过储备池被非线性映射到高维空间
- 高维空间中的特征更容易线性分离
- 训练只在输出层做一次线性回归，速度快、不涉及反向传播

### 1.3 为什么适合小样本预测？

RC 的核心优势恰好对应 CPI 预测的痛点：

- **训练参数少**：只有输出层权重需要训练，不需要大量数据
- **不需要反向传播**：避免了深度学习在小样本上的过拟合问题
- **天然适合时序**：储备池的动力学响应自带短期记忆，适合捕捉 CPI 的时序依赖

## 2. 核心概念

### 2.1 Echo State Network（ESN）

ESN 是 RC 最经典的实现形式。结构分三层：

```
输入层(u) → 储备池(x) → 输出层(y)
              ↑____________|
              反馈连接（可选）
```

**关键公式**——储备池状态更新：

$$x(t) = (1 - \alpha) \cdot x(t-1) + \alpha \cdot f(W_{in} \cdot u(t) + W \cdot x(t-1))$$

其中：
- $x(t)$：时刻 $t$ 的储备池状态向量，维度为 $N$（节点数）
- $u(t)$：时刻 $t$ 的输入
- $W_{in}$：输入权重矩阵，随机生成后固定
- $W$：储备池内部连接矩阵，随机生成后固定
- $\alpha$：泄露率（leak rate），控制状态更新的快慢
- $f$：非线性激活函数，通常用 $\tanh$

**输出公式**：

$$y(t) = W_{out} \cdot x(t)$$

$W_{out}$ 是唯一需要训练的矩阵，通过岭回归求解：

$$W_{out} = Y_{target} \cdot X^T \cdot (X \cdot X^T + \beta I)^{-1}$$

其中 $\beta$ 是正则化系数。

### 2.2 谱半径（Spectral Radius）

**定义**：储备池连接矩阵 $W$ 的最大特征值的绝对值，记作 $\rho(W)$。

**为什么重要**：谱半径决定了储备池的"记忆能力"和"稳定性"之间的平衡。

- $\rho(W) < 1$：系统稳定，输入的影响会逐渐衰减——这就是"回声状态性质"（Echo State Property）
- $\rho(W)$ 接近 1：记忆时间长，但接近不稳定边缘
- $\rho(W)$ 太小：记忆太短，无法捕捉长期依赖
- 通常设置在 0.8~0.99 之间

**实际操作**：随机生成 $W$ 后，计算其最大特征值，然后缩放：

$$W \leftarrow W \cdot \frac{\rho_{target}}{\rho(W)}$$

### 2.3 泄露率（Leak Rate）

**定义**：状态更新公式中的 $\alpha \in (0, 1]$，控制储备池状态更新的速度。

- $\alpha$ 小（如 0.1）：旧状态保留多，变化慢，适合捕捉慢变趋势
- $\alpha$ 大（如 1.0）：旧状态几乎被完全替换，响应快

**对 CPI 预测的意义**：CPI 是月度数据，变化缓慢，较小的 $\alpha$（0.3~0.5）通常更合适。

### 2.4 输入缩放（Input Scaling）

输入信号在进入储备池前需要乘以一个缩放因子。如果输入太大，储备池会饱和（所有节点输出接近 $\pm 1$）；太小则储备池响应太弱。通常设为 0.1~1.0。

## 3. 延时储备池（Delay-based RC）

### 3.1 核心思想

Appeltant 等人在 2011 年 Nature Communications 上提出：传统 RC 需要大量物理节点，硬件实现复杂。他们发现，一个带延时反馈的非线性节点就能替代整个储备池网络。

### 3.2 虚节点（Virtual Nodes）

在延时反馈环中，将延迟时间 $\tau$ 等分为 $N$ 个间隔，每个间隔 $\theta = \tau / N$ 处采样一次，这 $N$ 个采样点就是"虚节点"。

```
延迟线: |--θ--|--θ--|--θ--|--θ--| ... |--θ--|
虚节点:  x1    x2    x3    x4   ...   xN
         ←────────── τ ──────────→
```

虚节点之间的耦合来自非线性节点的动力学惯性——因为 $\theta$ 小于节点的特征响应时间，相邻虚节点会相互影响，形成类似传统 RC 中节点互联的效果。

### 3.3 输入掩码（Mask）

输入数据在进入延时环之前，需要乘以一个掩码矩阵 $M$：

$$J(t) = M \times I(t)$$

掩码的作用是让不同的虚节点"看到"输入信号的不同加权版本，增加状态的多样性。掩码通常随机生成（如从 $\{-1, +1\}$ 中均匀采样）。

### 3.4 Mackey-Glass 非线性节点

Appeltant 2011 使用的非线性节点是 Mackey-Glass 振荡器：

$$\dot{x}(t) = -x(t) + \frac{\eta \cdot (x(t-\tau) + \gamma \cdot J(t))}{1 + (x(t-\tau) + \gamma \cdot J(t))^p}$$

- $\eta$：反馈增益（类似 ESN 的谱半径）
- $\gamma$：输入增益（类似 ESN 的输入缩放）
- $p$：非线性指数
- $\tau$：延迟时间

### 3.5 延时 RC vs 传统 ESN

| 特性 | 传统 ESN | 延时 RC |
|------|----------|---------|
| 节点数 | 受限于计算资源 | 仅受限于采样速度 |
| 硬件实现 | 需要大量物理节点 | 只需一个非线性节点 |
| 拓扑结构 | 可自由设计 | 由延迟环决定（环形） |
| 训练方式 | 线性回归 | 线性回归 |
| 本项目优先级 | 第3阶段实现 | 第4阶段实现 |

## 4. 半导体激光器 RC

### 4.1 物理原理

半导体激光器在外部扰动（光反馈、光注入）下会表现出丰富的非线性动力学行为，包括稳态、周期振荡、混沌等。这些非线性响应天然适合作为储备池。

### 4.2 L-K 速率方程

描述光注入半导体激光器的动力学（扩展的 Lang-Kobayashi 方程）：

$$\frac{dE}{dt} = \frac{1}{2}(1+i\alpha)\left[\frac{g(N(t)-N_0)}{1+\varepsilon|E(t)|^2} - \frac{1}{\tau_p}\right]E(t) + kE(t-\tau)e^{-i\omega\tau} + k_{inj}E_{inj}(t)e^{i\Delta\omega t}$$

$$\frac{dN}{dt} = J - \frac{N(t)}{\tau_s} - \frac{g(N(t)-N_0)}{1+\varepsilon|E(t)|^2}|E(t)|^2$$

- $E$：光场复振幅，$|E|^2$ 即光强
- $N$：载流子密度
- $\alpha$：线宽增强因子
- $k$：反馈强度
- $k_{inj}$：注入强度
- $\tau$：反馈延迟

### 4.3 工作点选择

并非所有非线性状态都适合 RC。最佳工作点通常在注入锁定边界附近——系统既不是过于稳定（记忆能力不足），也不是完全混沌（可重复性差）。

## 5. RC 参数汇总表

| 参数 | 符号 | 典型范围 | 对 CPI 预测的建议 |
|------|------|----------|-------------------|
| 储备池节点数 | $N$ | 50~1000 | 先用 100~200 |
| 谱半径 | $\rho$ | 0.8~0.99 | 先用 0.9 |
| 泄露率 | $\alpha$ | 0.1~1.0 | 先用 0.3，CPI 变化慢 |
| 输入缩放 | $\gamma$ | 0.1~1.0 | 先用 0.5 |
| 正则化系数 | $\beta$ | $10^{-8}$~$10^{-2}$ | 先用 $10^{-4}$ |
| 稀疏度 | $s$ | 0.01~0.1 | 先用 0.05（5% 连接） |
| 窗口大小 | $w$ | 6/12/24 | 先用 12 |
| 虚节点间隔 | $\theta$ | — | 第4阶段再定 |
